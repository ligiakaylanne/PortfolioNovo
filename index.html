Jornada para o Sucesso no ENEM

O Projeto ENEM é uma iniciativa que desenvolvi com o objetivo de oferecer suporte educacional para estudantes de escolas públicas que não têm acesso a cursinhos preparatórios para o Exame Nacional do Ensino Médio (ENEM).

Utilizando HTML, CSS e JavaScript, criei uma plataforma online interativa e gratuita, que disponibiliza uma variedade de recursos educacionais, incluindo videoaulas, e dicas de estudo.
